# YOLOv8 Custom Training & Vehicle Detection

A real-time vehicle detection system built using YOLOv8, OpenCV, and Python. The project detects vehicles such as cars, buses, trucks, and motorcycles from images, videos, and webcam streams.

---

## Features

- Vehicle Detection using YOLOv8
- Detect Cars, Buses, Trucks, and Motorcycles
- Image Detection
- Video Detection
- Webcam Detection
- Bounding Boxes with Confidence Scores
- Fast Real-Time Inference
- Easy to Train with Custom Dataset

---

## Technologies Used

- Python
- YOLOv8 (Ultralytics)
- OpenCV
- NumPy

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/YOLOv8-Custom-Training-Deployment.git
```

Move into project

```bash
cd YOLOv8-Custom-Training-Deployment
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

## Usage

Run the application

```bash
python app.py
```

If using Streamlit

```bash
streamlit run app.py
```
